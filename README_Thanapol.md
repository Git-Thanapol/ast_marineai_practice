# Real-Time Application README

## Prerequisites

- Python 3.8

## Installation

Install required dependencies:

```bash
pip install -r requirements.txt
```

## Running the Application

Choose and run any of the following modules:

### battles.py
```bash
python battles.py
```

### comparison.py
```bash
python comparison.py
```

### console_ast.py
```bash
python console_ast.py
```

### dashboard_ast.py
```bash
python dashboard_ast.py
```

Each module can be run independently based on your needs.
